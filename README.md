# Mali Drought Monitoring Dashboard

## Project Overview

This project develops an interactive drought monitoring dashboard for Mali using rainfall and vegetation data.

The dashboard integrates climate indicators, drought indices, and GIS visualization to support drought monitoring and decision-making.

## Objectives

- Monitor drought conditions in Mali.
- Analyze rainfall variability.
- Compute drought indicators.
- Visualize drought trends interactively.

## Datasets

- CHIRPS Rainfall Dataset
- NDVI Dataset
- Mali Administrative Boundaries

## Drought Indicators

- Standardized Precipitation Index (SPI)
- Vegetation Condition Index (VCI)
- Rainfall Anomalies
- NDVI Anomalies

## Tools and Technologies

- Python
- Pandas
- GeoPandas
- Streamlit
- Plotly
- Folium
- GitHub

## Features

- Interactive drought dashboard
- Rainfall trend visualization
- SPI monitoring
- Drought category analysis
- GIS-based visualization
